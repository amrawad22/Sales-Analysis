# 📊 Sales Analysis Project

## Overview
This project analyzes sales data from the Superstore dataset to uncover valuable business insights and support data-driven decision-making.

The analysis focuses on:
- Sales trends over time
- Product and category performance
- Regional sales distribution
- Customer return behavior
- The impact of discounts on profitability

The project includes a Jupyter Notebook for data analysis, an interactive Tableau dashboard, and a presentation summarizing the main findings and recommendations.

---

## Files

### 📓 Notebook — `Sales_Analysis.ipynb`
This Jupyter Notebook includes:
- Data cleaning and preprocessing
- Handling duplicates, missing values, and outliers
- Data merging between sales and returns datasets
- Exploratory Data Analysis (EDA)
- Visualizations for sales trends, regional performance, and product analysis

---

### 📊 Dashboard — `Sales Analysis Dashboard`
Interactive Tableau dashboard featuring:
- Monthly and yearly sales trends
- Top-selling products and category analysis
- Regional sales and profit comparison
- Discount and shipping cost impact on profitability

---

### 📑 Presentation — `Sales_Analysis_Presentation.pdf`
Presentation summarizing:
- Data preprocessing workflow
- Customer return analysis
- Sales insights across regions and categories
- Business recommendations for inventory, shipping, and discount optimization

---

## Key Insights

- Sales peak during year-end months, while summer shows lower activity.
- The Western region generates the highest sales and profits.
- The Southern region underperforms compared to other regions.
- Furniture and Technology categories have relatively high return rates.
- High discounts negatively impact profitability.
